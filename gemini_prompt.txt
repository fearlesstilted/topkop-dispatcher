Привет. Мы совместно с тобой ведём проект TOPKOP RAG Dispatcher — AI-чат-виджет для строительной компании TOP KOP (Гołdap, Польша). Бот собирает лиды (услуга, объём, локация, телефон) и передаёт их руководству через Make.com.

Стек: Python 3.10, Gradio 6, AsyncOpenAI (Groq LLaMA-3), Make.com вебхук, база знаний KnowledgeTopKop.json.

Архитектура уже реализована и работает:
- Context Stuffing: весь KB (бетон, prefabrykaty, услуги) вшивается в системный промпт как минифицированный JSON
- Anti-hallucination guard в промпте
- Async везде (AsyncOpenAI, httpx)
- Sentry мониторинг
- Dead letter logging для потерянных лидов
- 8-польный маркетинг-парсер (включая телефон)
- Репо: github.com/fearlesstilted/topkop-dispatcher
